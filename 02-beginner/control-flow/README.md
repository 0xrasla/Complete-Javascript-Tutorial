# Control Flow

## What Is Control Flow?

Control flow is the order in which your code runs. By default, JavaScript runs code from top to bottom, line by line. Control flow statements let you change that — they let your code make decisions (if this, do that) and repeat actions (do this 10 times).

---

## Types of Control Flow

| Type | Keywords | Purpose |
|------|----------|---------|
| **Conditional** | `if`, `else if`, `else`, `switch` | Make decisions based on conditions |
| **Loop** | `for`, `while`, `do...while` | Repeat an action multiple times |
| **Loop Control** | `break`, `exit` a loop early |
| **Loop Control** | `continue` | Skip the current iteration and move to the next |
