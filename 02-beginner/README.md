# 02 — Beginner

Welcome to the beginner section. This is where you learn JavaScript from absolute zero. Every concept is explained in plain language, with code examples you can run immediately.

---

## Prerequisites

**None.** You do not need any prior programming experience. If you can use a web browser, you can learn from here.

---

## What You Will Learn

| # | Topic | Folder | Description |
|---|-------|--------|-------------|
| 01 | Variables | `01-variables/` | How to store and use data |
| 02 | Data Types | `02-data-types/` | The different kinds of values JavaScript supports |
| 03 | Operators | `03-operators/` | How to combine and compare values |
| 04 | Control Flow | `04-control-flow/` | How to make decisions and repeat actions |
| 05 | Functions | `05-functions/` | How to write reusable blocks of code |
| 06 | DOM Basics | `06-dom-basics/` | How JavaScript interacts with web pages |
| 07 | Projects | `07-projects/` | Hands-on projects to practice what you learned |

---

## How to Use This Section

1. Go through the folders in order (01 → 02 → 03 → ... → 07).
2. Read the `README.md` in each folder first — it explains the concept with code examples.
3. Open the `.js` file and run it in the browser console.
4. Change values and experiment.
5. Complete the projects at the end.
