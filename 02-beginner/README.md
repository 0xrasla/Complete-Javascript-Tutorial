# 02 — Beginner

Welcome to the beginner section. This is where you learn JavaScript from absolute zero.

---

## Prerequisites

**None.** You do not need any prior programming experience. If you can use a web browser, you can learn from here.

---

## What You Will Learn

| Topic | Folder | Description |
|-------|--------|-------------|
| Variables | `variables/` | How to store and use data |
| Data Types | `data-types/` | The different kinds of values JavaScript supports |
| Operators | `operators/` | How to combine and compare values |
| Control Flow | `control-flow/` | How to make decisions and repeat actions |
| Functions | `functions/` | How to write reusable blocks of code |
| DOM Basics | `dom-basics/` | How JavaScript interacts with web pages |
| Projects | `projects/` | Hands-on projects to practice what you learned |

---

## How to Use This Section

1. Go through the folders in order (variables → data-types → operators → control-flow → functions → dom-basics).
2. Read the `README.md` in each folder first.
3. Open the `.js` file and run it in the browser console.
4. Change values and experiment.
5. Complete the projects at the end.

---
