# Beginner Projects

## Practice Makes Permanent

The best way to learn is by doing. These projects let you apply everything you learned in the beginner section. Each project includes a complete, working solution with comments explaining every decision.

---

## Projects

| Project | Description | Skills Practiced |
|---------|-------------|-----------------|
| `tip-calculator.js` | Calculate tips and split bills | Variables, functions, arithmetic, user input |
| `number-guessing-game.js` | A simple number guessing game | Math, loops, conditionals, random numbers |
| `palindrome-checker.js` | Check if a word reads the same forwards and backwards | Strings, loops, conditionals, functions |

---

## How to Use

1. Read the README in this folder.
2. Try to build the project yourself first (even if it is messy).
3. Then read the solution to see how an experienced developer approaches it.
4. Modify the solution — add features, change behavior, make it yours.
