# Functions

## What Is a Function?

A function is a reusable block of code that performs a specific task. You define it once, give it a name, and then "call" it whenever you need it. Functions help you avoid repeating yourself and keep your code organized.

Think of a function as a recipe: it has instructions (the code inside), it can take ingredients (parameters), and it can give you a result (return value).

---

## Key Concepts

| Concept | Description |
|---------|-------------|
| **Declaration** | Creating a function with a name |
| **Parameters** | Variables listed in the function definition (the inputs) |
| **Arguments** | Actual values passed to the function when you call it |
| **Return Value** | The result the function gives back |
| **Arrow Function** | A shorter, modern syntax for writing functions |
| **Scope** | Where variables inside a function can be accessed |
