# 04 — Advanced

Welcome to the advanced section. This is where you go beyond writing code and start understanding how JavaScript actually works under the hood.

---

## Prerequisites

**You should already know:**
- Variables, data types, operators, control flow
- Functions (declarations, arrow functions, closures basics)
- Object-Oriented Programming (classes, inheritance, `this`)
- Asynchronous JavaScript (callbacks, Promises, async/await)
- Basic DOM manipulation

If any of these feel unfamiliar, go back to the [Intermediate section](../03-intermediate-oop/) first.

---

## What You Will Learn

| # | Topic | Folder | Description |

| 01 | Closures | `01-closures/` | Deep dive into closures, lexical scope, and practical applications |
| 02 | Memory & Engine | `02-memory-and-engine/` | How JavaScript manages memory, garbage collection, and how the engine works |
| 03 | Design Patterns | `03-design-patterns/` | Common solutions to recurring design problems |
| 04 | Projects | `04-projects/` | Hands-on projects to practice advanced concepts |

---

## How to Use This Section

1. Go through the folders in order (01 → 02 → 03 → 04).
2. Read the `README.md` in each folder first — it explains the concept with code examples.
3. Open the `.js` file and run it in the browser console.
4. Modify values and experiment.
5. Complete the projects at the end.

---

## A Word of Warning

Advanced topics are complex by nature. It is completely normal to not understand something on the first read. Re-read the explanations, run the code, change values, and break things. That is how you learn.
