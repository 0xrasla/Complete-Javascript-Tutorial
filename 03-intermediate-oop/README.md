# 03 — Intermediate: Object-Oriented Programming & Async JS

Welcome to the intermediate section. This is where you move from writing simple scripts to thinking about software design and handling real-world complexity.

---

## Prerequisites

**You should already know:**
- Variables (`const`, `let`), data types, and operators
- Control flow (`if/else`, `for`, `while`)
- Functions (declarations, arrow functions, parameters, return values)
- Basic DOM manipulation

If any of these topics feel unfamiliar, go back to the [Beginner section](../02-beginner/) first.

---

## What You Will Learn

| # | Topic | Folder | Description |
|---|-------|--------|-------------|
| 01 | Classes & OOP | `01-classes-and-oop/` | Classes, constructors, inheritance, `this`, encapsulation |
| 02 | Async JS | `02-async-js/` | Callbacks, Promises, async/await, the event loop |
| 03 | Projects | `03-projects/` | Hands-on projects to practice intermediate concepts |
| 04 | Modules | `04-modules/` | import/export, organizing code across files, dynamic imports |

---

## How to Use This Section

1. Go through the folders in order (01 → 02 → 03).
2. Read the `README.md` in each folder first — it explains the concept with code examples.
3. Open the `.js` file and run it in the browser console.
4. Modify values and experiment.
5. Complete the projects at the end.
