# 05 — Common Projects

These are practical, real-world projects that can be built at any level. They combine concepts from across the course and are great for practice.

---

## Projects

| Project | Description | Difficulty |
|---------|-------------|------------|
| `calculator.js` | A functional calculator with a clean interface | Beginner |
| `todo-list.js` | A full-featured to-do list with persistence | Beginner-Intermediate |
| `quiz-app.js` | An interactive quiz application | Intermediate |
| `pomodoro-timer.js` | A productivity timer based on the Pomodoro technique | Intermediate |

---

## How to Use

1. Pick a project that interests you.
2. Try to build it yourself first (even if it is messy).
3. Then read the solution to compare approaches.
4. Modify the solution — add features, fix bugs, make it yours.

---

## Note

These projects are designed to run in the browser. Some include a simple HTML interface, while others are console-based. Check the README in each project folder for setup instructions.
